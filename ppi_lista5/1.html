O que é JavaScript?

A motivação para a crianção JavaScript foi devido a falta de interatividade e dinamismo nas páginas web.

A execução no navegador ocorre no dispositivo do usuário, sendo rápida e interativa. Já no servidor, ocorre remotamente, ideal para tarefas complexas como processar dados.

No contexto dos navegadores da web, a sandbox é um mecanismo de segurança que isola a atividade de cada site, protegendo o sistema principal contra código malicioso.

document permite acessar e modificar o conteúdo da página web. Já o console é usado para exibir mensagens no console do navegador, útil para depuração. E por fim, alert exibe uma caixa de mensagem ao usuário.
